# To-do List API

## Descrição
Este é um projeto de uma API para gerenciar uma lista de tarefas (to-do list).

## Funcionalidades
- Criar uma nova tarefa
- Listar todas as tarefas
- Atualizar uma tarefa existente
- Excluir uma tarefa

## Tecnologias utilizadas
- Node.js
- Express.js
- H2 database
